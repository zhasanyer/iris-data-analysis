# Iris Dataset Mini-Project

## What I did
- Loaded the Iris dataset using scikit-learn
- Converted it to a Pandas DataFrame with readable species names
- Computed summary statistics (mean, std, min/max) for each feature
- Created 2 visualizations: a histogram of sepal length, and a scatter plot of petal length vs width

## What I found
- The dataset has 150 samples, 50 per species (setosa, versicolor, virginica)
- Petal length and width are the most useful features for separating species — setosa is clearly distinct, while versicolor and virginica overlap slightly
